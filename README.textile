Simple site for a small group of people to share urls.

* looks up url titles
* automatically adds thumbnails for and embeds Flickr, Imageshack, Vimeo and YouTube urls
* embeds Flash mp3 player for mp3 urls
* supports Gravatar
* Atom feed
* regex search
* looks good on iPhone
* rack interface

See "http://urls.matthewm.boedicker.org/":http://urls.matthewm.boedicker.org/ for example.

Setup:

<pre>
<code>
rake gemspec build
gem install pkg/murlsh-x.x.x.gem
</code>
</pre>

In web root:

<pre>
<code>
edit config.yaml
rake db:init user:add
</code>
</pre>
